  Configuration variables and directives (only valid in config files or with the -D switch)

@INCLUDECFG cfg@

Include a config file.

@DEFAULT_HEAD=hname@

Sets the default header without the template extension and directory (default 'head').

@DEFAULT_TAIL=tname@

Set the default footer without the template extension and directory (default 'tail').

@DEFAULT_TMPLDIR=tdir@

Sets the templates directory (default 'templates').

@DEFAULT_OUTPUTDIR=odir@

Sets the output directory (default '..'), if you want to use the same directory for .raw and .html files sets '@DEFAULT_OUTPUTDIR=.@'.

@DEFAULT_EXTENSION=ext@

Sets the extension for the output files (default 'html'), the dot is added internally by WPP.

@DEFAULT_RAW_EXTENSION=ext@

Sets the extension for the input files (default 'raw'), the dot is added internally by WPP.

@DEFAULT_TMPL_EXTENSION=ext@

Sets the extension for the template files (default 'tmpl'), the dot is added internally by WPP.
