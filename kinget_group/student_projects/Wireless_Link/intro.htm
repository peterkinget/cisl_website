2.1  Overall system description

 

The system consists of clients and server. The Client will find out its location from GPS signal and report its location to the server. (DGPS is optional for more accurate location need) Once the server receive the location report from Clients, server will send back the acknowledge signal.

 

2.2  Client: 

 

2.2.1       Normal Mode:

  • Report GPS Location to Server periodically.

  • Report period is decided by batter power.

  • If it doesn¡¦t catch the ACK signal from Server, it will boost the transmitting power.

 

2.2.2       Warning Mode

  • If it doesn¡¦t receive ACK for a certain period of time, it enters warning Mode.

  • A Pulse beacon and location is sent. The report period is decided by battery power.         

 

2.2.3       Go Home Mode

2.2.4       Emergency Mode

  • When Emergency Mode is activated, client will send HELP signal to server. The Report period will be shorter and is transmitted with max power.
