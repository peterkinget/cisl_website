Advanced Projects Opportunity



Study of the Implementation of a Time-Encoding Machine


If you are interested in the design of analog integrated circuits, and you have a strong background in signal processing, this project offers you the opportunity to learn about novel principles in signal processing for encoding analog signals.


The project starts with the evaluation of the practical implementability of a novel signal encoding algorithm using analog building blocks. The effect of circuit non-idealities on the properties of the encoding will be studied using extensive circuit and system simulations.


Depending on the available time the project can include the complete design and layout of a time encoder and the submission for fabrication. IC fabrication typically takes 12 weeks. The experimental verification of the chip can be done in a follow-up project.
