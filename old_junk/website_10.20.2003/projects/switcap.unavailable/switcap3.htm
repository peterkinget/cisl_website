Noise analysis of switched-capacitor networks using Switcap

The syntax for the network netlist is the same as described in the manual for Switcap2, with addition of several new features. (add link to new features)

For an example see the file  filter.scn

To run the noise analysis type:  sw3 < input_file_name

The results will be stored in a binary form in the file sw3.noise.

To view the results it is necessary to use the post processor psw3. Use  psw3 -h to get help on using the post processor. Currently, it prints the results in regular ASCII table format and in xgraph format.
