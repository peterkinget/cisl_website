You should read through this material before you attempt to use HSPICE.  The [Introduction] provides an overview of HSPICE, which is intended to give you an idea of what HSPICE can and cannot do for you.

When HSPICE is run from the command line, the circuit is to be described in the source file, and the results will be provided in a number of output files.  The above link [Files] provides a quick overview of what each filed is used for.

The link [Source File] provides information of how to write the source file, and details of how to describe [Devices] and [Commands] are provided in the respective links.

Finally, you may want to read the [Notes].
